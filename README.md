# HentAI API

The HentAI API code. Based on [example-api](https://github.com/zukijourney/example-api) (made by me, too).

## Dependencies

- Python 3.9+
- MongoDB database
- Redis database (rate limiting)
- A fucking brain

## How to Run

```bash
pip install -r requirements.txt -U
chmod +x run.sh
./run.sh
```
